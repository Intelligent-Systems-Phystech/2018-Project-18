\abstract
    {В работе по имеющемуся описанию сигнала моделируется поведение субъекта. Предлагается построить модель, решающую задачу декодирования сигналов ECoG. В качестве этапов построения такой системы решаются задачи предобработки данных, выделения признакового пространства и снижения размерности. Кроме того, декодирование сигналов должно происходить с минимальной задержкой, поэтому выбирается модель оптимальной сложности. В работе учитывается комплексная природа сигнала: непрерывная траектория движения, наличие дискретных структурных переменных, наличие непрерывных переменных.

\section{Введение}
{Нейрокомпьютерные интерфейсы (BCI) обеспечивают взаимодействие между мозгом и компьютером \cite{nicolas2012brain}. Данное взаимодействие в свою очередь позволит людям с ограниченными возможностями вернуть их мобильность при помощи бионических протезов. Для этого необходимо обеспечить точное и быстрое интерпретирование данных, получаемых с мозга.

Для получения данных мозга с целью последующего анализа для предсказания намерений часто используются электро- и магнитоэнцефалография \cite{waldert2008hand}\cite{wang2009predicting}\cite{quandt2012single}. Однако, электрокография позволяет получить измерения с большим разрешением во временной и пространственной областях. Имеется множество работ, показывающих применимость ECoG для определения движения, выполняемого объектом исследования, например, для предсказания движения руки \cite{gomez2010epidural}\cite{chao2010long}.

Из-за природы данных ECoG, признаковое пространство обладает огромной размерностью, что значительно усложняет построение модели для предсказания намерений. Существует множество методов снижения размерности признакового пространства, например, метод наименьших частных квадратов (Partial Least Squares -- PLS) \cite{geladi1986partial}. Но данные ECoG оказывается удобно представлять в пространственно-временно-частотной области, что делает применение PLS затруднительным. Однако, возможно использовать модификацию PLS -- многоразмерный метод наименьших частных квадратов (Multi-Way Partial Least Squares -- NPLS) \cite{eliseyev2011iterative}\cite{eliseyev2012l1}.

В данной работе проводится построение модели предсказания движений субъекта, позволяющей декодировать ECoG сигналы со снижением размерности данных способом, упомянутым выше. Кроме того, учитывается комплексная природа вывода модели: непрерывная траектория движения, наличие дискретных структурных переменных, наличие непрерывных переменных. Исследование проводится на основе данных из библиотеки сигналов человеческого мозна ECoG \cite{data2016}. }
